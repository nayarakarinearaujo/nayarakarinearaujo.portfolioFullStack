Projeto - API Pizzaria

Descrição do Projeto

Este projeto é uma aplicação web desenvolvida usando o framework Spring Boot. Ele utiliza um modelo entidade-relacionamento (ER) fornecido para realizar operações CRUD (Create, Read, Update, Delete) em pelo menos 4 entidades. A estrutura do projeto inclui:

  • Modelos: Classes Java que representam as entidades do modelo ER, com atributos mapeados para colunas no banco de dados e anotadas com @Entity.
  • Repositórios: Interfaces que estendem JpaRepository para fornecer métodos CRUD para cada entidade, incluindo salvar, buscar, atualizar e excluir.
  • Controladores: Classes responsáveis por manipular requisições HTTP relacionadas a cada entidade, utilizando anotações como @RestController e mapeando métodos para operações CRUD com @GetMapping, @PostMapping, @PutMapping e @DeleteMapping.

💡 Funcionalidades Principais:

  • Realizar operações CRUD em pelo menos 4 entidades do modelo ER fornecido.
  • Integrar classes Java como modelos de entidades (@Entity) para interação com o Spring.
  • Utilizar interfaces estendidas do JpaRepository para fornecer métodos de CRUD para cada entidade.
  • Mapear endpoints HTTP em classes controladoras (@RestController) para manipular operações CRUD.

🛠️ Tecnologias Utilizadas:

  • Java
  • Spring Boot
  • Banco de Dados - MySQL
  • GitHub
  • Postman
